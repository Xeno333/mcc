# mcc (This is a Work In Progress, NOT STABLE)
MIT licensed Compiler Collection

## Build on Linux
### Prepare:
For Ubuntu users:
```
	sudo apt install git make
```

### Get source:
```
	git clone https://github.com/Xeno333/mcc.git
```

### Build:
```
	cd mcc
	make
```
Now the mcc package should be located in bin/pkg
